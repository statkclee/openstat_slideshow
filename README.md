# 발표 안내

발표는 두가지 방식으로 진행됩니다.

- 실시간 라이브
- 동영상 녹화


# 발표에 도움이 되는 내용

- 실시간 라이브 코딩을 강추합니다 !!!
- 16:9 화면비를 추천
- 멀리서도 잘 보일 수 있는 큰 폰트 
- R 코드를 가독성 있는 변수명과 코드 사이 간격과 공백도 적절히 분배하도록 신경 쓴다.
- 데모 코드에 사용되는 팩키지명을 명시한다.
